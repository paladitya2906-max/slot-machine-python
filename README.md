# Slot Machine Game (Python)
A command-line based slot machine game built using Python.
This project simulates real slot machine mechanics with betting,spinning, and winnings calculation.
## Features
Deposit money system
Betting on multiple lines
Random slot generation
Winnings calculation logic
Input validation for safe user interaction
## Technologies used 
Python
## How to Run
Download the project
Open terminal/command prompt
Run: python SlotMachine.py
